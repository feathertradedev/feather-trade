Before using Feather, make sure you control the selected wallet and understand the assets you intend to trade or supply.

<Steps>
1. Install and unlock a supported wallet.
2. Select the intended account and Robinhood Chain.
3. Keep ETH available for network fees.
4. Verify token addresses, not symbols alone.
5. Read the pool address, bin step, current price, and fee information.
6. Review every approval and transaction in the wallet before signing.
</Steps>

## Use small amounts first

If a token, pool, strategy, or wallet workflow is new to you, begin with an amount you can afford to lose. Confirm the resulting balances and activity before increasing exposure.

<Warning title="Feather never needs your recovery phrase">
Do not enter a seed phrase or private key into the application, documentation, support messages, or vulnerability reports.
</Warning>
