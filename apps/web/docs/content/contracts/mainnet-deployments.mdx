## Verification policy

This page never publishes testnet, local development, dry-run, zero, or placeholder addresses as mainnet contracts. Integrators should compare addresses against the official explorer and the currently selected release before signing or encoding transactions.

<Risk title="Do not preconfigure placeholder addresses">
Wait for verified mainnet values. A syntactically valid address can still target the wrong chain or contract.
</Risk>
