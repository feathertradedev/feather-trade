Feather uses Liquidity Book v2.2 contracts on Robinhood Chain. Verified mainnet integration details will publish after deployment validation.

## Contract roles

- **LBFactory** registers pairs, presets, quote assets, and supported implementations.
- **LBPair** holds reserves, executes bin-level swaps, and mints or burns liquidity balances.
- **LBToken** defines fungible multi-token balances keyed by bin ID.
- **LBRouter** prepares user-facing swap and liquidity operations across supported pairs.
- **LBQuoter** computes candidate exact-input paths without executing them.
