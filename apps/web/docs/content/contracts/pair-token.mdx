LBPair holds market reserves and maps liquidity to bin IDs. LBToken behavior exposes each owner's fungible balance in those bins.

## Required identity checks

Validate the factory, token X, token Y, bin step, hook configuration, deployed code, and active ID before preparing a write. Treat a deep link or visually matching symbol as untrusted until these checks agree.

## Position semantics

An owner can hold balances across many bin IDs. Overlapping deposits merge. Removing liquidity burns selected balances and returns proportional current reserves. There is no unique position NFT and no independent LP fee claim.
